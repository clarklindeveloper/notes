UNDERSTANDING AND TYPING ‘THIS’

the this keyword - this keyword will change depending on how we invoke the function

——————————————————

//function
function myFunction(){
console.log(“FUNCTION: ”, this); }
myFunction(); //window is invoking the function …this points to Window


——————————————————

//object literal

const myObj = {
myMethod(){
console.log(‘OBJECT: ’, this);  } };
myObj.myMethod(); //myObj invokes object… ‘this’ points to myObj
——————————————————

//class

class MyClass{
myMethod(){
console.log(‘Class: ‘, this);
}
}

const myInstance = new MyClass();
myInstance.myMethod(); //intance invokes myMethod..this points to myClass
